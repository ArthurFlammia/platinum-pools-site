# Platinum Pools — MP1: Website for Someone Else

A simple, mobile-friendly website for Platinum Pools, an owner-operated
pool service company in Rowley, MA serving the Massachusetts North Shore.

**Live site:** https://arthurflammia.github.io/platinum-pools-site/

## The Client

Platinum Pools (Vernon, owner) — a local pool company offering weekly
service, seasonal openings/closings, water testing and chemical
balancing, repairs, and pool installation. Their existing web presence
was a basic WordPress page, so the goal was a cleaner, phone-first site
that matches how they actually work with customers (they prefer calls
over email).

## Pages

- `index.html` — home: hero, about, services teaser, service area
- `services.html` — weekly service detail plus seasonal and repair services
- `contact.html` — phone-first contact info and service area

## Built With

- Semantic HTML (`header`, `nav`, `main`, `section`, `footer`)
- External CSS with Flexbox layout and a mobile media query
- Deployed on GitHub Pages
